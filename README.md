# Chrome-Extension
## How to use
1. git clone this repository
2. service-worker.js 파일의 27번째 줄의 주소를 자동 답변 생성 프로젝트의 EC2 인스턴스 주소로 변경합니다.
3. access chrome://extensions/
4. load the tutorial.websockets file
